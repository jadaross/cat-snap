# Competitive Analysis

A survey of existing apps in the street cat / community cat mapping space.

---

## Existing Apps

### Whisker Tracker
**URL:** whiskertrackerapp.com | **Platforms:** iOS, Android

The closest existing competitor. Whisker Tracker is designed specifically for cat walkers, photographers, and colony caretakers.

**Features:**
- GPS-based cat cataloging with facial recognition
- Full cat profiles with photo history
- Public comment "wall" per cat
- Community sharing with neighbors
- Map view of all found cats

**Gaps / weaknesses:**
- Heavy focus on colony caretaking / TNR workflows
- UI appears utilitarian rather than consumer-friendly
- Social features are basic (comments only, no friend graph)
- No friend/social feed — it's a shared map, not a social network

---

### Purry
**URL:** fifilaw.com/purry | **Platforms:** iOS

A cat habitat tracking app aimed at TNR volunteers and community caretakers.

**Features:**
- Photo + notes + location per cat
- TNR status tracking (trapped, neutered, returned)
- Badge/gamification system for tracking milestones
- Neighborhood stray cat population view

**Gaps / weaknesses:**
- Volunteer/advocacy focused, not casual-consumer
- No social feed or friend system
- Limited discovery/exploration features

---

### StraySync
**App Store:** StraySync | **Platforms:** iOS

Find stray cats and dogs; community rescue coordination.

**Features:**
- Stray animal sighting map
- Focused on rescue coordination

**Gaps / weaknesses:**
- Focused on rescue/adoption, not ongoing neighborhood cat tracking
- Dogs and cats mixed — not cat-specific
- No individual cat profiles or check-in mechanic

---

### Meow Mapper
**URL:** meow-mapper.github.io | **Platforms:** Web

A community project/prototype for mapping cat sightings.

**Gaps / weaknesses:**
- Appears to be a prototype / side project
- No social features
- Limited polish and ongoing development unclear

---

### Catalog (Cat Welfare Society prototype)
**Reference:** [Medium writeup by Mark Choo](https://medium.com/@markchoo/catalog-an-app-to-track-stray-community-cats-e6cb291bf09a)

A two-week design prototype built for the Cat Welfare Society.

**Features:**
- Cat profiles with last-seen status
- Location history per cat
- Photo uploads

**Gaps / weaknesses:**
- Prototype only, not a shipped product
- No social layer

---

### FeralScan
**URL:** feralscan.org.au | **Platforms:** Web, limited mobile

Australian government-backed feral animal monitoring tool, includes feral cats.

**Gaps / weaknesses:**
- Focused on conservation/pest management, not neighborhood community cats
- Not consumer-friendly
- Bureaucratic UX

---

### I Know Where Your Cat Lives
**URL:** iknowwhereyourcatlives.com

A data art project that maps ~7 million public cat photos using GPS metadata scraped from Instagram and Flickr. Not an app — more of a privacy awareness project.

**Notable:** Demonstrates the appetite people have for cat photo maps. Went viral.

## Summary: Market Gap

| Feature | Whisker Tracker | Purry | Cat Snap (planned) |
|---------|----------------|-------|-------------------|
| Map of sightings | ✓ | ✓ | ✓ |
| Cat profiles | ✓ | Partial | ✓ |
| Photo per sighting | ✓ | ✓ | ✓ |
| Friend social graph | — | — | ✓ |
| Social feed | — | — | ✓ |
| Cat check-ins | — | — | ✓ |
| Reactions / comments | Basic | — | ✓ |
| Consumer UX focus | Partial | — | ✓ |
| Cat recognition/linking | ✓ (facial AI) | — | Planned v3 |
| iOS + Web | ✓ | iOS only | Web now, iOS planned |

**Cat Snap's differentiation:**
1. **Social-first design** — a friend graph and activity feed make it feel like "street cat Instagram" rather than a volunteer tool
2. **Consumer UX** — designed for casual cat spotters, not TNR coordinators
3. **Check-in culture** — the "is Felix still around?" mechanic encourages return visits and community bonds
4. **Individual cat identity** — emphasis on naming, describing, and following specific cats builds emotional attachment
